# BREACH Frontend

React + Vite UI for the BREACH platform.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Environment

Set `VITE_API_URL` to point to backend:
```bash
VITE_API_URL=http://localhost:8000 npm run dev
```

For production, this is auto-configured by Render.
